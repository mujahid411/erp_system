# ERP System

ERP System is a web application designed to provide businesses with a comprehensive dashboard and management tools for handling orders, products, and schedules. It is built using React and Tailwind CSS.

## Features

- **Dashboard**: Provides an overview of key business metrics and insights.
- **Orders**: Allows users to view, manage, and track orders efficiently.
- **Products**: Enables management of product inventory, descriptions, and pricing.
- **Calendar**: Provides a visual representation of scheduled orders and events.

## Technologies Used

- React: JavaScript library for building user interfaces.
- Tailwind CSS: Utility-first CSS framework for styling web applications.
- React Router: Declarative routing for React applications.

## Getting Started

To run the ERP System locally, follow these steps:

### Clone the repository
git clone https://github.com/mujahid411/erp_system.git

### Navigate to the project directory
cd erp_system

### Install dependencies
npm install

### Start the development server
npm start

Open your browser and navigate to `http://localhost:3000`
